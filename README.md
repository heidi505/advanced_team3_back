## 📓 주요 기능 설명
#### 회원가입
- 구글 API를 사용하여 이메일 연동

#### 채팅 기능
- 실시간 채팅, 그룹채팅
 
#### 앨범 & 카메라 기능
- 카메라 및 앨범 연결해서 사진 보내기
 
#### 프로필 기능
- 프로필 배경과 프로필 이미지 변경
- 닉네임, 상태메시지 변경

#### 채팅방 메뉴 기능
- 즐겨찾기 추가
- 상단 고정
- 이름 바꾸기
- 공지사항
